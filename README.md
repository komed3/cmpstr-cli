# CmpStr CLI

**cmpstr-cli** is a Node.js CLI wrapper for the [CmpStr](https://www.npmjs.com/package/cmpstr) package. It enables direct use of CmpStrâ€™s core features such as string normalization, similarity scoring, and matrix comparison via the terminal.

## Installation

You can install the tool globally via NPM:

```bash
npm install -g cmpstr-cli
```

Or run it directly via npx (no installation required):

```bash
npx cmpstr-cli
```
